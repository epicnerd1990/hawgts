## HAWGTS Icon Pack

The icon package that comes with HAWGTS mostly contains Material design icons, along with some modified specifically for use in this widget. Material is the icon style used in Home Assistant. The icons can be entered anywhere in the Kustom Editor when prompted to choose an icon. All licence information at [the bottom](#licence-information) for these icons.

Name Syntax: `[source-]` `<materialname>` `[-mod]`  
`[source-]`: [MDI](), [Material](), custom  
`<materialname>`: Name given by  designers  
`[-mod]`: MDI or Material icon modified to enhance the widget design
